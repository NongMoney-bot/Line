สำหรับการสมัครใช้งาน NGR Billing Bot ผ่าน LINE LIFF

สายงานตลาดค้าปลีกและวิศวกรรมก๊าซธรรมชาติ บริษัท ปตท. จำกัด (มหาชน) ("บริษัท") ให้ความสำคัญกับการคุ้มครองข้อมูลส่วนบุคคลของท่าน และขอแจ้งรายละเอียดเกี่ยวกับการเก็บรวบรวม ใช้ และเปิดเผยข้อมูลส่วนบุคคลของท่าน ดังนี้

1. ข้อมูลส่วนบุคคลที่เก็บรวบรวม
เมื่อท่านสมัครใช้บริการ NGR Billing Bot ผ่าน LINE LIFF บริษัทอาจเก็บรวบรวมข้อมูลดังต่อไปนี้
  • ชื่อ-นามสกุล
  • LINE User ID
  • Display Name จากบัญชี LINE
  • เบอร์โทรศัพท์
  • อีเมล
  • ชื่อบริษัท/หน่วยงาน (ถ้ามี)
  • ข้อมูลแพ็กเกจที่สมัคร
  • ประวัติการสมัครและการใช้งานบริการ
  • ข้อมูลการชำระเงินหรือเอกสารที่เกี่ยวข้องกับการสมัครใช้บริการ
2. วัตถุประสงค์ในการประมวลผลข้อมูล
บริษัทจะใช้ข้อมูลส่วนบุคคลของท่านเพื่อ
  • สมัครและจัดการสิทธิการใช้งาน Subscription Package
  • ยืนยันตัวตนและเชื่อมโยงบัญชีผู้ใช้งานกับระบบ NGR Billing Bot
  • ติดต่อประสานงาน แจ้งเตือน และให้บริการหลังการขาย
  • จัดทำใบแจ้งหนี้ ใบเสร็จรับเงิน และเอกสารทางธุรกิจที่เกี่ยวข้อง
  • ปรับปรุง พัฒนา และเพิ่มประสิทธิภาพการให้บริการ
  • ปฏิบัติตามกฎหมายหรือข้อกำหนดที่เกี่ยวข้อง
3. ฐานกฎหมายในการประมวลผลข้อมูล
บริษัทประมวลผลข้อมูลส่วนบุคคลของท่านตามฐานกฎหมายดังต่อไปนี้
  • เพื่อการปฏิบัติตามสัญญา (Contract)
  • เพื่อประโยชน์โดยชอบด้วยกฎหมายของบริษัท (Legitimate Interests)
  • เพื่อปฏิบัติตามกฎหมาย (Legal Obligation)
  • ความยินยอมของท่าน (Consent) ในกรณีที่กฎหมายกำหนด
4. การเปิดเผยข้อมูล
บริษัทอาจเปิดเผยข้อมูลส่วนบุคคลของท่านแก่
  • ผู้ให้บริการระบบเทคโนโลยีสารสนเทศ
  • ผู้ให้บริการชำระเงิน
  • ผู้ให้บริการ Cloud Infrastructure
  • หน่วยงานภาครัฐหรือหน่วยงานที่มีอำนาจตามกฎหมาย
โดยบริษัทจะดำเนินการเท่าที่จำเป็นและมีมาตรการคุ้มครองข้อมูลที่เหมาะสม

5. ระยะเวลาการเก็บรักษาข้อมูล
บริษัทจะเก็บรักษาข้อมูลส่วนบุคคลของท่านตลอดระยะเวลาที่ท่านใช้บริการ และหลังสิ้นสุดการใช้บริการตามระยะเวลาที่กฎหมายกำหนด หรือเท่าที่จำเป็นสำหรับวัตถุประสงค์ที่ระบุไว้

6. สิทธิของเจ้าของข้อมูล
ท่านมีสิทธิตามกฎหมาย PDPA ได้แก่
  • สิทธิขอเข้าถึงข้อมูล
  • สิทธิขอแก้ไขข้อมูล
  • สิทธิขอลบข้อมูล
  • สิทธิขอจำกัดการประมวลผล
  • สิทธิคัดค้านการประมวลผล
  • สิทธิถอนความยินยอม (กรณีอาศัยฐานความยินยอม)
7. ช่องทางติดต่อ
หากท่านมีข้อสงสัยเกี่ยวกับการคุ้มครองข้อมูลส่วนบุคคล สามารถติดต่อ
